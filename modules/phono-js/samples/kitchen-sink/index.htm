The Kitchen Sink

The kitchen sink sample showcases all of the functionality available to you in the Phono SDK. With it, you can create multiple phono instances; each able to make and receive multiple calls and chats. The log viewer, located at the bottom of this sample, monitors client/server events and errors. You can make calls to any Phono session ID, US domestic phone numbers, as well as any Tropo or Voxeo Evolution application. You can learn more about how to use Phone by visiting the documentation page and the forums.
